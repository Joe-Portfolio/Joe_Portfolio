# 프로젝트 분석 보고서: Joe_Portfolio

이 문서는 `Joe_Portfolio` 프로젝트의 소스 코드 분석 결과를 담고 있습니다.

## 1. 프로젝트 개요
이 프로젝트는 **시니어 게임 기획자 및 서비스 매니저인 조현우(CHW)** 님의 개인 포트폴리오 웹사이트입니다. React와 Vite를 기반으로 구축되었으며, 모던하고 인터랙티브한 UI를 통해 경력과 프로젝트를 효과적으로 보여줍니다.

## 2. 기술 스택
- **Framework**: React 18.2.0
- **Build Tool**: Vite 6.2.0
- **Language**: TypeScript (TSX)
- **Styling**: Tailwind CSS (App.tsx의 class 명칭을 통해 확인)
- **AI Integration**: `@google/genai` (Google Gemini AI API 연동 준비/사용)

## 3. 주요 프로젝트 구조
- `App.tsx`: 애플리케이션의 메인 엔트리 포인트. 다국어(한국어/영어) 지원 및 레이아웃 구성.
- `data.ts`: 모든 프로젝트 정보, 경력 사항, 기술 스택 데이터가 중앙 관리되는 파일.
- `components/`: UI를 구성하는 재사용 가능한 컴포포넌트들.
  - `Header.tsx`, `Hero.tsx`, `Projects.tsx`, `Experience.tsx`, `Skills.tsx` 등.
  - `InteractiveBackground.tsx`: 배경에 동적인 효과 제공.
  - `AIGallery.tsx`: 생성형 AI를 활용한 크리에이티브 워크플로우 전시.
- `index.html`: 앱의 기본 HTML 템플릿.

## 4. 핵심 기능 분석
- **다국어 지원**: 브라우저 언어 설정을 감지하여 한국어와 영어를 자동으로 전환합니다 (`App.tsx`의 `useEffect`).
- **인터랙티브 UI**: 비디오 모달, 인터랙티브 배경, AI 갤러리 등의 시각적 요소를 통해 사용자 경험을 강화했습니다.
- **데이터 중심 구조**: `data.ts` 파일만 수정하여 포트폴리오 내용을 쉽게 업데이트할 수 있도록 설계되었습니다.
- **AI 활용**: 단순한 텍스트 기획을 넘어 생성형 AI(Stable Diffusion, Gemini 등)를 실무 프로세스(Sketch to AI)에 도입한 사례를 강조합니다.
